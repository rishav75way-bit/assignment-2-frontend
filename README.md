# Assignment 2 – Frontend (Rooms + Synced Video Player)

React UI for joining a room as a host or viewer. Host uploads a video and controls play/pause/seek; all viewers stay synced via sockets.

## Tech
- React + TypeScript (Vite)
- Tailwind CSS
- Socket.IO Client
- Axios (API calls)

## Setup
npm install

Create .env in frontend root:

VITE_API_BASE_URL=http://localhost:3000
